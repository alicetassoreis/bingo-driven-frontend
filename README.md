## Frontend - Bingo Driven

Interface do usuário para o sistema de bingo.


## Tecnologias
- React
- Vite
- TypeScript
- Styled-components
- Docker


## Docker

docker build -t bingo-frontend .

docker run -p 3000:3000 bingo-frontend


## Docker Compose

docker-compose up --build

ou:

docker-compose up -d --build

docker-compose down


## Execução local

npm install
npm run dev
